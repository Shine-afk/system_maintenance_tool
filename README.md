# System Maintenance Tool

Лёгкая утилита для диагностики и обслуживания системы.

## Возможности

- Сбор базовой информации о системе (ОС, процессор, память, диск)
- Выполнение дополнительных проверок (опционально)
- Кроссплатформенность (Linux / Windows)
- Минимальные зависимости
- Очистка системного мусора

## Установка

Просто склонируйте репозиторий:

```bash
git clone https://github.com/Shine-afk/system-maintenance-tool.git
cd system-maintenance-tool
```

## Использование

### Быстрый старт
```bash
python3 system_maintenance_tool.py -c
```

### Базовая диагностика
```bash
python3 system_maintenance_tool.py
```

### Очистка системного мусора

Для лучшей производительности системы изпользуйте модуль очистки мусора.

```bash
python3 system_maintenance_tool.py --clear
```

Для очистки используется clear.so, если необходимо пересобрать clear.so используйте build.sh и clear.py
