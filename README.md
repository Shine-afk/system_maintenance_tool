# System Maintenance Tool

Инструмент для обслуживания и проверки системы Ubuntu.

Инструмент поддерживает два основных режима работы:
- Проверка текущего состояния системы
- Применение обновлений из локального кэша

Зашифрованные данные обновлений хранятся в отдельном файле `apt-update.cache`.
